# Hệ Thống Quản Lý Nhà Trọ / Ký Túc Xá

## Mô tả
Hệ thống giúp quản lý phòng trọ hoặc ký túc xá, bao gồm quản lý phòng, người thuê, hợp đồng và thanh toán tiền phòng.
Mục tiêu là giúp chủ trọ hoặc ban quản lý dễ dàng theo dõi tình trạng phòng và lịch sử thanh toán một cách hiệu quả.

## Công nghệ sử dụng
- Backend: Spring Boot
- Frontend: Thymeleaf
- Database: PostgreSQL
- Version Control: Git / GitHub
- API Testing: Postman

## Cài đặt và chạy 

## Yêu cầu
- Java 17+
- Node.js 18+ (nếu dùng React)
- PostgreSQL

## Chạy Backend
cd backend
./mvnw spring-boot:run
